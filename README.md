# STUNIVOZ

Full-Stack Student Career Platform with 23+ modules built with:
- React 18 + TypeScript + Vite + Tailwind CSS
- Firebase (Auth, Firestore, Storage)
- Admin & Recruiter Panels
- Responsive Design

## 🚀 Quick Start

```bash
npm install
npm run dev
```

**Dev Server**: http://localhost:5173
